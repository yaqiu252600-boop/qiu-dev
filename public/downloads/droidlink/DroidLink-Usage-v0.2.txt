DroidLink 本地互传 V0.2
========================

本版包含 Windows 程序和 Android 原生 APK。

安装
----
1. Windows：双击 DroidLink.exe，防火墙提示时只允许“专用网络”。
2. Android：安装 DroidLink-Android-v0.2.apk；建议 Android 10 或更高版本。

连接
----
1. 手机和电脑连接同一个 Wi-Fi。
2. 打开 Windows 端。
3. 打开 Android App，点击“扫描二维码”。
4. 在手机授权框选择“仅本次允许”或“信任此电脑”。

传文件
------
- 手机到电脑：在 Android App 点击“选择手机文件”，支持多选。
- 电脑到手机：在 Windows 端选择或拖入文件，然后在手机中刷新并下载。
- 文件夹：Windows 端会自动压缩成 ZIP 后发送。

保存位置
--------
- Android：Download/DroidLink/
- Windows：用户目录\Downloads\DroidLink，可在界面更改。

安全
----
- 文件只在局域网中直传，不经过云端。
- 配对码一次性使用，连接后使用随机令牌。
- 手机和电脑均可主动断开连接。

暂未实现
--------
USB、蓝牙、投屏、鼠标键盘控制、剪贴板同步和手机文件管理将在后续阶段继续开发。
