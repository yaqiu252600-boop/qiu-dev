# 高考志愿真实数据接入说明

当前已完成真实数据系统架构和来源登记，但尚未导入任何可确认的官方录取数据文件。

已接入：

- 全国来源入口登记：阳光高考院校库、阳光高考专业知识库、阳光志愿信息服务系统。
- 样例省份来源登记：广东省教育考试院。
- processed 目录中的结构化演示数据已明确标记 `isDemo: true`，不得作为真实招生计划或录取结果使用。

仍需人工下载或核验：

- 广东省教育考试院发布的历年录取分数、投档线、一分一段表、招生计划 PDF / Excel / 公告。
- 阳光高考或高校官网公开的院校库、专业库、招生章程数据。
- 其他省级教育考试院或招生考试信息网的官方入口和文件下载地址。

接入要求：

- 每条真实数据必须包含 `sourceName`、`sourceUrl`、`year`、`province`、`updatedAt`。
- 不采集需要登录、付费、验证码绕过或违反 robots / 使用条款的数据。
- 不复制商业平台、付费平台、闭源项目数据。
